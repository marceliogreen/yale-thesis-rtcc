# RTCC Thesis Downloads

This folder contains downloaded datasets for the RTCC thesis analysis.

## Contents

- `washington_post_homicides.csv` - 52,179 homicide records with clearance data (2007-2017)
- `rtcc_homicide_washington_post.csv` - Processed data for 8 RTCC cities
- `chicago_homicide_full.csv` - Chicago homicide counts 2010-2023
